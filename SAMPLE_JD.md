# Sample Job Description for Testing

## Software Engineer - Frontend Developer

**Company:** TechCorp Inc.

**Role:** Frontend Developer

**Job Description:**

We are looking for a talented Frontend Developer to join our dynamic team. The ideal candidate will have strong experience in modern web development technologies and a passion for creating exceptional user experiences.

### Requirements:

- Strong proficiency in JavaScript, TypeScript, and React
- Experience with Next.js framework
- Solid understanding of data structures and algorithms (DSA)
- Knowledge of RESTful APIs and GraphQL
- Experience with SQL databases (PostgreSQL preferred)
- Familiarity with MongoDB for NoSQL solutions
- Understanding of AWS cloud services
- Experience with Docker and CI/CD pipelines
- Knowledge of testing frameworks like Jest and Cypress
- Strong problem-solving skills and ability to work in a team environment
- Excellent communication skills

### Responsibilities:

- Develop and maintain responsive web applications using React and Next.js
- Design and implement efficient database queries using SQL
- Deploy applications using Docker containers on AWS infrastructure
- Write unit tests and integration tests using Jest and Cypress
- Collaborate with backend developers to integrate REST and GraphQL APIs
- Optimize application performance and ensure scalability
- Participate in code reviews and follow best practices

### Nice to Have:

- Experience with Kubernetes
- Knowledge of Linux/Unix systems
- Understanding of system design principles

---

**This JD contains keywords for:**
- Languages: JavaScript, TypeScript
- Web: React, Next.js, REST, GraphQL
- Data: SQL, PostgreSQL, MongoDB
- Cloud/DevOps: AWS, Docker, CI/CD, Kubernetes, Linux
- Testing: Jest, Cypress
- Core CS: DSA
